# eth_getBlockTransactionCountByHash
---
## Case
  
geth --http --dev

only one transaction per block
  
### Call

``
{
	"jsonrpc":"2.0",
	"method":"eth_getBlockTransactionCountByHash",
	"params":[
		"0x0f664288f277da61285151568654cf92d518146c76759a1cfec5d4b06324e4e2"
	],
	"id":1
}
`` 

### Return

``
{
    "jsonrpc": "2.0",
    "id": 1,
    "result": "0x1"
}
``
